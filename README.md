# CyberPurge
CyberPurge can kill all your apps running in your background through two methods (Assesibilty/Shizuku)

It is designed with much minimilastic & futuristic UI design

It works fine in both modes but i prefer to use shizuku method to kill apps at lightning speeds

It doesn't closes any system level apps, it only closes third party apps running in background 

There is an option to exclude apps from list in settings

**Disclaimer:- The assesibilty metjod is tested only in One UI, I am not 100% sure about other operating systems, but the shizuku method works on every operating system seamlessly jod
